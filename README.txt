# 📱 SwiftBar Plugin: scrcpy Launcher (by Muhamad Prasetyo)

Plugin ini memungkinkan kamu untuk menjalankan `scrcpy` dari menu bar macOS menggunakan [SwiftBar](https://github.com/swiftbar/SwiftBar). Support koneksi wireless otomatis dan reconnect cepat!

## ✨ Fitur
- Jalankan `scrcpy` via USB dan WiFi (dengan auto-connect).
- Reconnect ke IP device terakhir.
- Reset ADB dan hapus data IP.
- Refresh otomatis setiap 5 menit.
- Notifikasi suara & popup macOS.
- Dibuat oleh **Muhamad Prasetyo**.


🔗 Syarat

    macOS

    SwiftBar (https://swiftbar.app)

    scrcpy

    adb (via android-platform-tools)
